# Anagram Text Converter
The functional text formatter that reflects input text into an anagram-style format

## Features

- Reverses the letters of each word independently
- Preserves non-letter symbols located at the beginning or end of each word
- Retains the original spacing and text formatting
- Processes arbitrary input provided via the console

## Example

Input:
account1 balance:4500USD

Output:
tnuocca1 DSUecna:4500lab
